All *italic* text is code.

Create a new virtual environment called, "quant"

*conda create -n quant*





Activate the new quant environment in the anaconda prompt.

*conda activate quant*



Packages need to be installed inside of the Prompt first before they can be installed inside of code.

As an example, we are installing the "Pandas", "NumPy", "Matplotlib", "Datetime" packages.

*conda install pandas*

*conda install numpy*

*conda install matplotlib*

*conda install datetime*
