The Python_YouTube project is designed to help people learn how to program in Python for statistics, machine learning,
model development, and model validation. A lot of the content will focus on quantitative finance however the ideas
will be focus on developing math and statistical models (this includes machine learnings).

These videos are NOT designed for a computer scientist, quant dev (model implementation), or other software engineering
roles. "Quants" are those building models and most of our coding is more similar to scripting and not software engineering!
